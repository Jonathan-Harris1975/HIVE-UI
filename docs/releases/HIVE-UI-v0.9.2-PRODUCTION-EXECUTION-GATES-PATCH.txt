HIVE-UI v0.9.2 production execution gates patch
=================================================

Goal
----
Align the HIVE-UI Operations screen with the HIVE backend v1.25 production execution-gates behaviour.

Changes
-------
- Replaced the hard-coded "execution adapters remain disabled" banner text with /health-driven adapter policy status.
- Added execution adapter readiness to the Operations health flags.
- Added typed support for execution_adapters_enabled and execution_adapter_policy in HealthResponse.
- Added typed support for adapter_execution_enabled, execution_state and execution_mode in ExecutionReviewItem.
- Changed controlled execution preview badges so approved, executable previews show "Ready for handoff" rather than a blocked-only badge.
- Added an Execution column to the review queue so approved reviews can show ready_for_execution.
- Updated StatusBadge and WorkflowGraph handling for ready_for_execution.
- Updated API/session documentation to describe approved, allow-listed production handoff behaviour.
- Bumped HIVE-UI to 0.9.2 / production-execution-gates.
