# HIVE-UI v0.8.0

This coordinated frontend release requires the matching HIVE multi-bucket backend contract.

## Delivered

- Full authenticated read access to configured R2 lanes.
- Upload/write controls restricted to the HIVE uploads lane.
- Prefix browsing, search, pagination, metadata, preview and download.
- Shared-chat attachment links include the source lane.
- Source citations display the originating lane.
- Searchable grouped model picker with Auto route first.
- Discovery-only image and video generation categories.
- Canonical signed-session token validation.
- Updated CI actions and preserved source-secret checks.

## Cloudflare Pages variables

No new HIVE-UI variables are required. Continue using:

```text
HIVE_API_BASE_URL
HIVE_ADMIN_TOKEN
HIVE_UI_ACCESS_KEY
HIVE_UI_SESSION_TTL_SECONDS
```
